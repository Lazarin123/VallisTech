## ⚡ Sobre a Vallis Tech

A **Vallis Tech** não é apenas uma software house; somos o seu braço tecnológico. Nascemos diretamente das trincheiras da tecnologia, unindo uma mentalidade rigorosa de **Quality Assurance (QA)** com a paixão pelo desenvolvimento web moderno e escalável. 

Nossa cultura é baseada no **"Engineering First"** — construímos produtos robustos, livres de falhas críticas e com foco absoluto na conversão e experiência do usuário.

## 💻 O Portal Institucional

Este repositório reflete a nossa identidade digital. O site foi projetado do zero para ser uma vitrine de nossa capacidade técnica, aliando **alta performance, arquitetura limpa e design imersivo**.

### ✨ Features de Destaque:
- 🎨 **UI/UX Premium:** Paleta corporativa imersiva (Navy Blue, Cyan e Neon Green) projetada para transmitir confiança e inovação.
- 🏗️ **Arquitetura SCSS Modular:** Estilização componentizada utilizando a metodologia **BEM (Block, Element, Modifier)**.
- 📱 **100% Responsivo:** Experiência fluida e adaptável do Mobile ao Ultrawide.
- 🎬 **Micro-Interações Avançadas:** Animações on-scroll via *Intersection Observer* e expansão fluida de componentes construídos em Vanilla JS, focando em performance sem o peso de bibliotecas de terceiros.
- 🧩 **Componentes Estratégicos:** FAQ Acordeão dinâmico, portfólio interativo e painéis de código demonstrativos.
